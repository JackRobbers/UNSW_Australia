WikiMate

Check Required URLs for iGEM Wikis

A page is considered to be present if it has non-default content and the "Delete this box in order to be evaluated for this medal criterion" box has been removed for pages that have that. If a page is present but has not been capitalised correctly that will also be detected.

Disclaimer: This is an automated tool and maybe contain bugs, always double check your pages manually. This tool has been written for the 2020 version of the competition and may not work for previous or future years.

Enter your team name (e.g UNSW_Australia), year and any awards you intend to contest. If an award is not listed, it does not have a required url.
